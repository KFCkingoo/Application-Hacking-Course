# h1 Scope First, Controls Later
Tämä raportti on tehty keväällä Teron ja Larin 2026 Sovellusten hakkerointi ja haavoittuvuudet - ICI012AS3A-3003 kurssia varten.

## a) Scope
### 1. Soveltamisala
#### Kotiverkon perusinfra:
- Reititin
- VBox NAT
#### Kurssiharjoituksiin käytettävät laitteet:
-	PC ja läppäri
-	Linux koneet (Debian, Linux Mint) VBoxilla 
#### Tiedot:
-	Kurssimateriaalit
-	Laksu
-	GitHub repo
-	Harjoitusaineistot
-	Tunnukset/avaimet
-	Toisten tehtävänpalautukset

### 2. Ulosrajaukset:
Perheenjäsenten laitteet
  - Ei omistajuutta
  - Ei hallinnoitavissa
  - Ei merkitystä kurssille
    
Viihdelaitteet
  - Ei merkitystä kurssille
    
Kampuksen hallinnoima kone
  - Ei omistajuutta
  - Ei tule käyttöön
  - Kampus vastaa tietoturvasta
    
Operaattorin verkko "internetin puolella"
  - Ei hallinnoitavissa
  - ISP vastaa omasta verkosta

### 3. Keskeiset rajapinnat
Pilvipalvelut
  - GitHub
  - Moodle
  - Teams
    
Etäyhteydet
  - Reititin, palomuuri
  - Internet
  - NAT
  - ISP
  - Pilvipalvelut

Toimittajat
  - ISP
  - Laitetoimittajat (Lenovo, Intel, AMD)
  - Pilvipalvelintarjoajat (Microsoft, GitHub)
